About HORITAKE Tattoo Studio

葛飾金町刺青処　彫岳/HORITAKEは、お客様一人一人の人生を華やかにするきっかけを、「刺青」とゆう形で責任をもって提供させて頂きます。 どんなデザインでも、お客様のイメージより高いクオリティの作品の制作を心がけています。 アットホームなスタジオとなっていますので、緊張せずお越し頂けると幸いです。

手彫り　TEBORI

日本の太古から「彫りもの」として用いられる技法『手彫り』。 機械彫りとは違い、肌に入る質感は和紙に絵を描いたようなふわっとしたマッドな質感が特徴です。 墨ならではの馴染み方や色の発色の良さを生かした作品を作り上げる事を心掛けています。

機械彫り　MACHINE

現代では主流になっている機械彫り。 細かい部分の色入れや全体の筋彫り等、あらゆる表現をする事が可能です。 手彫りに比べるとツヤっとした質感が特徴です。機械にしか出せない表現、質感を追求しています。

TEBORI x MACHINE

僕のスタイルはマッドな質感の手彫り、細かい表現が可能な機械、二つの技法を用いて仕上げていきます。 それまであった古き良き和彫りの表現、今の時代だからこそ出来る新しい表現を織り交ぜて、唯一無二、独自の『彫もの」を追求しています。
